This is a port of the [React Native Tiny Quickstart](https://github.com/plaid/tiny-quickstart/tree/main/react_native) demo to an `expo` (CNG) app.

## Build and Run

### Setup

```bash
$ yarn setup --clean

```

### Start the server

```bash
$ node server.js

```

### Start the app

```bash
$ yarn android

```

or

```bash
$ yarn ios
```
